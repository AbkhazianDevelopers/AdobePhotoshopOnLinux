# AdobePhotoshopOnLinux

Выдаём права доступу установщику.

```bash
chmod +x install.bash
```

Запускаем (без суды)

```bash
bash install.bash
```

После установки, по пути `~/Desktop/adobephotoshop` будет лежать установщик AdobePhotoshop.
